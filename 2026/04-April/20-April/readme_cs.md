# 2078. Two Furthest Houses With Different Colors (C#)

## Understanding the Problem
We aim to maximize the difference between indices with unequal values.

## Plan
- Iterate through the array
- Compare each element with:
  - First element
  - Last element
- Use Math.Max to store result 

## C# Feature
Math.Max simplifies comparisons and keeps code clean.

## Complexity
Time Complexity: O(n)  
Space Complexity: O(1)
