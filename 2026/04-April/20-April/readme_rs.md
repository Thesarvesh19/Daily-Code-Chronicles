# 2078. Two Furthest Houses With Different Colors (Rust)

## Core Idea
Leverage Rust's iterator-style looping with strong type safety.

## Execution
- Iterate using index range
- Compare current value with boundaries
- Use max method for updates

## Rust Feature
Type casting is required when converting indices to i32.

## Complexity
Time Complexity: O(n)  
Space Complexity: O(1)
